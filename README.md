# Python Quiz Application 

A simple command-line quiz game in Python that test your knowledge on genertal topics and programming concepts.
The application ask multiple-choice questions, evaluates your answer, and gives a score with feedback.

# Features 

Multiple-choice questions (A/B/C/D format)

Real-time feedback for each answer (Correct/Wrong)

Score calculation at the end of the quiz

Performance-based feedback: Excellent, Good, or Needs Improvement

Easy to add or modify questions
